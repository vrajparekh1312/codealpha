# 🎧 MELO – Modern Web Music Player

MELO is a **modern, responsive web-based music player** built using **HTML, CSS, and JavaScript**.
It provides an elegant interface with playlist management, music controls, keyboard shortcuts, and a dynamic user experience.

The player allows users to browse tracks, search songs, manage favorites, and control playback with smooth UI interactions.

---

# 🚀 Features

## 🎵 Music Playback

* Play / Pause music
* Next / Previous track navigation
* Shuffle mode
* Repeat mode
* Progress bar with seeking

---

## 📂 Playlist Management

* Dynamic playlist rendering
* Track numbering
* Album artwork display
* Track duration display
* Active song highlight

---

## 🔍 Search System

Users can easily find songs using the search bar.

Supports search by:

* Song title
* Artist name

Search updates the playlist **in real time**.

---

## ❤️ Favorite Songs

Users can mark songs as **liked**.

Features:

* Favorite icon toggle
* Stored in **localStorage**
* Quick filtering of liked songs

---

## 🕒 Recently Played

The player tracks recently played songs.

* Displays recent tracks as quick access chips
* Stored using **localStorage**
* Maximum of **5 recent tracks**

---

## 🎨 Dynamic Background

The background color automatically changes based on the **album artwork colors**.

This creates a more immersive experience similar to modern music streaming platforms.

---

## 🎚 Interactive Music Controls

The player includes full playback controls:

* Shuffle button
* Repeat button
* Volume slider
* Play / Pause animation
* Ripple click animation

---

## 📊 Progress Tracking

* Real-time progress indicator
* Seek functionality
* Song time display (current / total duration)

---

## 📱 Mini Player

When the user scrolls down, a **mini music player** appears at the bottom.

Features:

* Compact playback controls
* Song information
* Progress indicator

---

## ⌨ Keyboard Shortcuts

Keyboard shortcuts allow faster control.

| Key   | Function        |
| ----- | --------------- |
| Space | Play / Pause    |
| →     | Next Song       |
| ←     | Previous Song   |
| ↑     | Increase Volume |
| ↓     | Decrease Volume |
| S     | Toggle Shuffle  |
| R     | Toggle Repeat   |

---

# 🎨 UI Features

The interface includes several modern UI enhancements:

* Glass-style layout
